- C programming language evolution is under continuous control
  - Backward compatibility
- Why evolve
  - Computer architecture has changed. New data types
  - Some language elements have proved to be inconvenient or confusing
  - New programming techniques have appeared in the meantime
- When the first version of C programming language were designed and implemented, its authors were focused on creating a tool that allowed the easy implementation of a new operating system
  - For PDP-11 computer by a company named Digital Equipment Corporation (DEC)
- PDP-11 was byte-oriented
- PDP-11 knew only two kinds of integer data: The byte (87 bits long) and the word (16 bits long)
- PDP-11 had a innovative instruction set
- C language was strongly influenced by ALGOL, PL/I (Programming Language One) and Fortran.
